
\section{Определение стоимости материальных затрат}\label{sec:opredelenie_stoimosti_mat_zatrat}

Система позволяет автоматически на основании введенных данных рассчтитать стоимость материалов, израсходованных на выполненный ремонт. Общая схема получения суммы прямых материальных затрат прведена на рисунке:

\begin{figure}[H]
\hspace*{-1.5cm}{\includegraphics[width=19cm]{pic/shema_opredelenia_pryamyh_zatrat}}
\caption{Схема получения прямых материальных затрат}
\label{ris:shema_opredelenia_pryamyh_zatrat}
\end{figure}

В таблице приведено описание шагов схемы.

\begin{longtable}[c]{|p{0.1\textwidth}|p{0.9\textwidth}|}
\hline 
Номер шага 	&Описание	\\ \hline 
\endfirsthead
\hline
1	&2	\\ \hline 
\endhead 
1& Подготовленный и утвержденный отчет по материальным затратам передается в систему бухгалтерского учета в виде документа. На основании данных отчета формируются записи о фактических затратах материалов в количественном выражении	\\ \hline 
2& Бухгалтер, ответственный за списание материалов заполняет документ требование-накладная в системе бухгалтреского учета на основании отчета по материальным затратам	\\ \hline 
3& На основании созданного документа требование-накладная создается документ <<Заявка на освоение>> для согласования аналитики и сумм списания затрат \\ \hline 
4& Заявка передается в систему управленческого учета для согласования \\ \hline
5& Данные согласованной заявки передаются в систему бухгалтерсого учета в виде распоряжений на принятие к учету рахсодов\\ \hline
6& Актуализация данных о затратах по документам распоряжения на принятие к учету рахсодов\\ \hline
7& Формирование данных о затратах на основании данных бухгалтерского учета\\ \hline
8& Передача данных о фактических затратах в систему управленческого учета\\ \hline
9& Формирование факта исполнения бюджета доходов и расходов на основании данных о затратах из системы бухгалтерского учета\\ \hline
10& Передача информации о затратах на ремонты (с заполненым реквизитом <<Вид работы ТОиР>>) в систему ТОиР для расчета стоимости материалов, израсходованных на ремонты и уточнения аналитики расхода\\ \hline
11& Определение коэффициентов для распределения стоимости списания номенклатуры по выполненным ремонтам за отчетный период\\ \hline
12& Полуение базы для распределения сумм списанных материалов для распределения на ремонты\\ \hline
13& Формирование данных о затратах с уточнением аналитики и распределенной по выполненным ремонтам за отчетный период\\ \hline
14& Передача данных о прямых материальных затратах на ремонты с уточненной управленческой аналитикой в систему управленческого учета. \\ \hline

\caption{Описание шагов схемы получения стоимости материальных затрат}
\label{tab:opisanie_shemy}
\end{longtable}
